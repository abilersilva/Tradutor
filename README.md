# Tradutor 🌍

Aplicação web simples para tradução de textos entre idiomas, com interface em **HTML/CSS** e integração com API de tradução.

## 🚀 Demonstração
Veja online: [Netlify Deploy](https://luminous-beignet-7644da.netlify.app)

## 📦 Tecnologias
- HTML
- CSS
- JavaScript
- API de tradução (ex.: LibreTranslate)

## 🛠️ Como usar
1. Clone este repositório:
   ```bash
   git clone https://github.com/abilersilva/Tradutor.git
   ```
